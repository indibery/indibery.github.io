공직플랜 (PlanRoutine)

초등 교사를 위한 iOS 업무 일정 관리 앱

앱 소개

공직플랜(PlanRoutine)은 한국 초등학교 교사를 대상으로 한 iOS 앱입니다. 매년 반복되는 학교 업무 사이클을 작년 데이터(생산문서등록대장 CSV)에서 가져와 올해 일정으로 빠르게 세팅하고, 일정을 검토·확정하면서 필요한 항목만 골라 Google 캘린더에 동기화할 수 있습니다.

iOS only 한국어 교사 업무 사이클 특화

핵심 기능

  • 작년 CSV 가져오기 — 생산문서등록대장 CSV에서 결재유형이 "생산"인 문서만 자동으로 골라 일정으로 변환합니다.
  • 일정 검토·확정 — 슬라이드 한 번으로 확정/삭제. 카테고리별 필터로 학생학적·일과운영 등 분류만 보고 검토할 수 있습니다.
  • 자체 캘린더 — 앱 내 캘린더에서 일정을 보고 직접 이벤트를 추가·완료 처리합니다.
  • Google 캘린더 동기화 (단방향) — 사용자가 명시적으로 오른쪽 슬라이드한 개별 이벤트만 사용자 본인의 Google 캘린더에 생성합니다. 기존 일정을 읽거나 수정·삭제하지 않습니다.
  • 기기 캘린더 동기화 — Apple/Android 시스템 캘린더로도 같은 방식으로 단방향 저장이 가능합니다.
  • 로컬 알림 — 월초·1주 전·1일 전 08:00에 시스템 알림.

Google API 사용 안내

본 앱은 사용자가 동의한 경우에만 Google Calendar API의 https://www.googleapis.com/auth/calendar.events scope을 사용합니다. 단방향 쓰기 전용이며, 사용자가 슬라이드 액션으로 승인한 단일 이벤트만 사용자 본인의 기본 캘린더에 생성·갱신합니다. Google 캘린더의 기존 일정을 읽거나 수정·삭제하지 않습니다.

Google 사용자 데이터 처리에 대한 자세한 내용은 개인정보처리방침의 §5(개인정보 제3자 제공) 항목을 참고해 주세요. 본 앱은 Google API Services User Data Policy의 Limited Use 요구사항을 준수합니다.
